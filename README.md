# telegram-trigger-bot

Телеграм-бот для групповых чатов, который реагирует только на строго заданные фразы.

## Поведение бота

В группах бот:

- На сообщение **`да`** отвечает: `кабзда!`
- На сообщение **`нет`** отвечает: `с-маркетинга ответ!`
- На сообщение **`300`** или **`триста`** отвечает: `)))))))))`

> Важно: бот реагирует **только** если всё сообщение целиком равно одной из этих фраз  
> (после приведения к нижнему регистру и обрезки пробелов).  
> Любые другие сообщения он игнорирует, чтобы не спамить чат.

## Локальный запуск

1. Создать бота через BotFather и получить токен.
2. Включить у бота:
   - Добавить в нужный чат.
   - В BotFather выключить *Group Privacy*, чтобы бот видел сообщения в группе.
3. Клонировать репозиторий:

   ```bash
   git clone <url-на-твой-репозиторий>
   cd telegram-trigger-bot
   ```

4. Создать и активировать виртуальное окружение (опционально):

   ```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   ```

5. Установить зависимости:

   ```bash
   pip install -r requirements.txt
   ```

6. Задать переменную окружения с токеном:

   ```bash
   export TELEGRAM_BOT_TOKEN="ТВОЙ_ТОКЕН"
   ```

7. Запустить:

   ```bash
   python -m app.bot
   ```

## Деплой на Render.com

В репозитории уже есть файл `render.yaml`.

1. Залей репозиторий в GitHub.
2. В Render создай новый **Worker** из этого репозитория.
3. Render сам подтянет:
   - `requirements.txt` для зависимостей,
   - `render.yaml` для конфигурации.
4. В настройках сервиса на Render добавь переменную окружения:

   - `TELEGRAM_BOT_TOKEN` → токен бота.

После деплоя добавь бота в нужные чаты — и всё, он будет реагировать только на фразы `да`, `нет`, `300`, `триста`.
